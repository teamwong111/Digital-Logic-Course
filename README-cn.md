## 简介
本仓库为我在2019年同济大学数字逻辑课程的大作业：基于FPGA 、蓝牙模块、VGA以及声音传感器的Flappy Bird小游戏

---

## 项目目录结构
```bash
.
├─report
│  ├─report.pdf
├─code
│  ├─bluetooth.v//蓝牙模块
│  ├─game.v //游戏模块
│  ├─key.v  //游戏按键模块
│  ├─sound.v//声音传感器模块
│  ├─top.v  //顶层模块，连接各个子模块
│  ├─tube.v //数码管模块
│  ├─vga.v  //VGA模块
│  └─top.xdc//管脚文件
```
